# GPT Image asset drop folder

Put GPT Image generated game assets here when generation is available.

Expected files:

- `heroes.png`: three original light heroes on a transparent background or chroma-key-removed PNG.
- `kaiju-sheet.png`: multiple original stage monsters on a transparent background.
- `backgrounds/level-01.png` through `backgrounds/level-26.png`: one 16:9 side-scrolling background per alphabet level.

The current game uses Canvas-drawn fallback characters, monsters, and stages so it remains playable even before these assets exist.
